# Flask Boiler Plate

A barebones flask application boilerplate

## Requirements
```bash
pip install -r requirements.txt
```

## Launch
```bash
git clone https://github.com/chnihy/flask_boiler_plate.git
cd flask_boiler_plate
python3 main.py
```
### Then open your browser and go to 
```
http://127.0.0.1:5000/
```
